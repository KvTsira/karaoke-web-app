# Karaoke Made Easy

## Purpose
To allow a Karaoke enthusiast to more easily search a song or artist and be presented with lyrics all in one application

## Frameworks implemented
- Skeleton
- Normalize
- jQuery

## Server Side APIs
- Genius API
- Lyrics.ovh API

## Built With
- HTML
- CSS
- Javascript
